## 🛠 Technical Stack
- **Frontend:** React (TypeScript), Tailwind CSS, Framer Motion
- **Animations:** HTML5 Canvas, Vanilla JS (Robust Injection Engine)
- **Deployment:** Vercel (CI/CD Pipeline)
- **SEO:** Schema.org JSON-LD, OpenGraph Metadata, Image Alt Tags
- **Optimization:** System.Drawing Image Compression (99% Reduction)

## 💎 Key Features
- **AI Technical Terminal:** Interactive "live" logs and command palette (`Ctrl + K`).
- **Data Snake Arcade:** Custom-built snake game hidden in the palette (`/snake`).
- **Analysis Engine:** Professional 25-year investment analysis tool (`/analyse`).
- **Custom UI:** Emerald-glow cursor and 3D magnetic profile photo.
- **Financial Pulse:** Stationary footer animation simulating real-time market data.
- **Matrix Mode:** Hidden background effect (Type `data` on keyboard).

## 📈 Scalability & Workflow
This project is built for rapid iteration:
1. **Updates:** Edit `index.html` or the `assets/` bundle for visual/logic changes.
2. **Push:** `git add .` -> `git commit -m "Update message"` -> `git push`.
3. **Automated Deployment:** Vercel automatically creates preview links and deploys the production build.
