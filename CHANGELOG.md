# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-22
### Added
- **AI Terminal:** Live typing logs and `/whois`, `/bash`, `/analyse` commands.
- **Snake Arcade:** Fully functional mini-game collecting "Data Bits."
- **Command Palette:** `Ctrl + K` power-user interface.
- **Matrix Easter Egg:** `data` keyboard trigger.
- **UI Enhancements:** Custom forest-green cursor, 3D magnetic profile photo, and background glow.
- **Financial Pulse:** Stationary live data ticker in the footer.
- **UX Improvements:** Back-to-Top button, smooth section scrolling, and mobile menu delay fix.
- **SEO & PWA:** Schema.org JSON-LD, OpenGraph tags, and mobile Home Screen support.

### Changed
- **Image Optimization:** Compressed photography from 73MB to ~1MB (99% reduction).
- **Location Sync:** Unified all mentions to Edinburgh, Scotland.
- **Contact Form:** Connected to Formspree for live lead generation.
- **Asset Paths:** Migrated to relative paths for bulletproof hosting.
